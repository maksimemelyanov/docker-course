# 🐳 Полезные ресурсы по Docker

Подборка книг, видео, курсов, документации и инструментов для изучения Docker (общий раздел + специально для C#/.NET).

## 📖 Книги по Docker (бесплатные и классические)

*   **Docker Deep Dive (Nigel Poulton)** – одна из лучших книг для начинающих и продолжающих.  
    [Amazon](https://www.amazon.com/Docker-Deep-Dive-Nigel-Poulton/dp/1916585252) | [Packt](https://www.packtpub.com/en-us/product/docker-deep-dive-9781835080715)
*   **The Docker Book (James Turnbull)** – классика, подробно про контейнеризацию.  
    [dockerbook.com](https://dockerbook.com/)
*   **Docker for Developers (русский перевод)** – бесплатный гайд от Docker.  
    [docs.docker.com/get-started](https://docs.docker.com/get-started/)
*   **Using Docker (Adrian Mouat)** – практическое руководство.  
    [O'Reilly](https://www.oreilly.com/library/view/using-docker/9781491915752/)

## 📦 Полезные Docker-образы для обучения

*   `hello-world` – минимальный образ для проверки.
*   `nginx:alpine` – легковесный веб-сервер.
*   `postgres:15-alpine` – PostgreSQL.
*   `redis:alpine` – Redis.
*   `rabbitmq:3-management-alpine` – RabbitMQ с веб-интерфейсом.
*   `mcr.microsoft.com/dotnet/sdk:8.0` – .NET SDK.
*   `mcr.microsoft.com/dotnet/aspnet:8.0` – .NET runtime для ASP.NET.

## 🎓 Рекомендуемый порядок изучения (для C# разработчиков)

1.  Пройдите официальный туториал [Docker Get Started](https://docs.docker.com/get-started/) (30 мин).
2.  Изучите [.NET Docker Tutorial от Microsoft](https://learn.microsoft.com/ru-ru/dotnet/core/docker/introduction).
3.  Выполните лабораторные работы из этого курса (3 работы по 40 мин).
4.  Посмотрите видео [Docker Tutorial for Beginners (Nana)](https://www.youtube.com/watch?v=3c-iBn73dDE).
5.  Изучите Docker Compose и многоконтейнерные приложения.
6.  Попробуйте [Play with Docker](https://labs.play-with-docker.com/) для практики без установки.
